Este proyecto consiste en simular un campeonato de motociclismo inspirado en MotoGP, utilizando programación orientada a objetos en Python. El usuario podrá interactuar desde la consola para definir el número de pilotos que participarán en el campeonato. A partir de una base de datos interna (usando diccionarios), se seleccionarán aleatoriamente los pilotos, hasta alcanzar el número deseado.

Cada piloto será representado como un objeto con distintos atributos. Algunas de las características estadísticas que se incluirán son, por ejemplo: el estado de ánimo del piloto en el día de la carrera, la calificación obtenida en la temporada anterior, su experiencia (años compitiendo), así como otras métricas personalizables.

La idea es que, una vez generados, se puedan visualizar estadísticas, comparar rendimientos y simular carreras. Además, se podría extender con nuevas funcionalidades, como configuraciones de circuito, condiciones meteorológicas, y la evolución de las carreras en cada evento.
